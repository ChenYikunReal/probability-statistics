# 概率论与数理统计学习资料分享

2019-2020秋季学期的考核方式和高等数学一样：
- 平时成绩10'
- 慕课成绩10'
- 单元测试15'
- 期中成绩15'
- 期末成绩50'

#

平时成绩和慕课成绩很可能合并<br/>
一次没交慕课作业好像扣3分<br/>
两次没互评相当于一次没交作业v
他们根本不按比例扣！！！<br/>

#

单元测试就是机考<br/>
方圆的题库貌似不能用了，以后就麻烦很多了<br/>
一共考三次：
- 1-2
- 3-4
- 6-8

# 

期中考试是[1-5]概率论考试

#

期末考试闭卷笔考<br/>
多做做课本的题，好好学，分特别高；不好好学，分很难看<br/>
考试都是大题，以后都是网阅卷，注意好好写字<br/>
大概6个大题左右，每个题都是分几个小问那种<br/>
且不说概率论学的不好怎么办，就算你概率会，只要不会数理统计一般会考的很难受，很多很多题不会做<br/>
不过这课学着没意思也是真的
